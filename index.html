About Me

Projects

Created a Tic-Tac-Toe game from scratch using HTML, CSS, JavaScript, Bootstrap, jQuery, and Ruby on Rails for the back-end. The game connects to an API created using Ruby on Rails that allows player to access their old games, scores, and a multiplayer feature. The game was created in four business days and the multiplayer feature uses Server Polling making it possible for two player to play at the same time.

Card game created using HTML, CSS, JavaScript, Bootstrap, jQuery, and Ruby on Rails for the back-end. This game uses the API to store the user data and also acts card as a dealer and allowing the player to play against the computer. The game was created in 5 business days and it allows a player make an account and CRUD the game resource.

Blog created in 3 business days using HTML, CSS, JavaScript, jQuery, Bootstrap, Mongoose, and the Express.js framework running on Node.js. The page allows users to CRUD a blog resource containing a title and a body, see and like posts from other users and also follow other users. Also the users can upload a picture and use it as their profile picture.

myPersonal Trainer is an application to create and keep track of exercise workouts. The user has the freedom to create new workouts by choosing from around 500 exercises available. In a future the will be capable of recording each user Best Records and compare data of each workout.
